# 获取航班列表

简要描述：

  • 根据时间获取航班列表

请求URL：

  • beta.xxxx.com/flight/get_list

请求方式：

  • GET/POST

参数：

参数名 必选 类型 说明
date 是 int 当天0点时间戳（1547171796）

返回示例：

{
  "status": true, // true:成功，false:失败/不符合要求
  // "code": 200, // 如果特殊需要再使用
  "msg": "请求成功",
  "data": [
    {
      "fid": "xxxxxxxx",
      "fnum": "MU1234",
      "status": 1,
      "completed": false
    },
    {
      "fid": "xxxxxxxx",
      "fnum": "CA5678",
      "status": 2,
      "completed": true
    },
    ...
  ]
}

注意

  1. 无论有没有数据，data的数据类型都不能变，如：data: [...]，没数据的时候不能变成 data: {}
  2. 字段数据类型也不能变，如：status: 1 不能变成 status: '1'

返回参数说明：

参数名 类型 说明
fid string 航班id
fnum string 航班号
status int 航班状态 1:起飞，2:到达，3:延误，4:备降，...
completed bool 是否完成 true:完成，false:未完成
