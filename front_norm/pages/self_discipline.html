# 自我约束

# 规范

定制规范自主遵守执行，方便大家，也方便自己。

# 测试

项目开发完成后，先自测，主动去优化，再提交测试。

# 时间控制

  • 上下班时间控制
  • 作息时间控制

# 请假

需要与组长及所负责项目的相关人员讲下

# 沟通

有问题多沟通，如果一时研究不出来，不要钻牛角尖，先做别的，或请教下别人，以免影响开发工期。

# 周报

每周周报要自觉添加，内容不要过少，注意日期范围。

# 项目缩略图

每完成一个项目都要上传对应的缩略图文件。

# API文档

如果是自己写的接口文档，要自主更新维护。
