课程内容
使用react cli创建项目，配合react-router、jquery、proxy代理做一个从登录到列表的增删改查及打包发布过程，外加一个简单的未登录验证
依赖包列表
                            "dependencies": {
                                "react": "^16.4.0",
                                "react-dom": "^16.4.0",
                                "react-scripts": "1.1.4",
                                "react-router": "^3.2.0",
                                "jquery": "^3.3.1"
                            },
                            "devDependencies": {
                                "mockjs": "^1.0.1-beta3"
                            }
要点
1、使用cli创建项目,安装依赖,运行
                                    npm install -g create-react-app

                                    create-react-app react_course
                                    react_course> npm i
                                    react_course> npm start
2、在下载模块的时候可以使用淘宝镜像
                                    npm install -g cnpm --registry=https://registry.npm.taobao.org

                                    cnpm i react-dom --save
3、生成配置目录

如果项目与git关联，先把git修改都提交了，再运行以下命令

                                    npm run eject
4、配置proxy代理

scripts/start.js 修改完成后重启服务

                                    let proxyConfig = prepareProxy(proxySetting, paths.appPublic);
                                    if(!proxyConfig){
                                        proxyConfig = {
                                            "/api/*":{
                                                target:"http://xxx.com/api",
                                                changeOrigin:true,
                                                secure:false
                                            }
                                        }
                                    }
5、加载本地数据件

粗暴解决方案，把data.js放到public目录下
